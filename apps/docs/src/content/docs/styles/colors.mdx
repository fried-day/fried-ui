## How to Use

```tsx
<button className="bg-primary text-primary-foreground">Solid</button>
<div className="bg-danger-pale text-danger-pale-foreground border border-danger-pale-border">
  Alert
</div>
```

## Dark Mode

All colors switch automatically via `.dark` class or `data-theme="dark"`. Toggle the theme on this page to see the difference.
