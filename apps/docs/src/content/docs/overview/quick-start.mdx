title: Quick Start
description: Install fried-ui and render your first component in under a minute.
---

## Installation

```bash
pnpm add @fried-ui/react @fried-ui/styles
```

Import styles in your CSS entry file:

```css
@import "tailwindcss";
@import "@fried-ui/styles";
```

## Usage

```tsx
import { Button } from "@fried-ui/react";

export default function App() {
  return <Button variant="primary">Click me</Button>;
}
```
