# @fried-ui/styles

Pure CSS design tokens and component styles for Fried UI. Built on [Tailwind CSS v4](https://tailwindcss.com/).

Zero JavaScript dependency — works with React, Vue, Svelte, or plain HTML.

## Installation

```bash
pnpm add @fried-ui/styles
```

Peer dependency: `tailwindcss >=4.0.0`

## Usage

Import styles in your CSS entry file:

```css
@import "tailwindcss";
@import "@fried-ui/styles";
```

## Development

```bash
pnpm dev     # Watch mode
pnpm build   # Build
```
